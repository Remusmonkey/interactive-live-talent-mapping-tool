# Interactive Live Talent Mapping Tool

A dynamic, shareable talent market brief built for executive searches — and other kinds of hiring too. Delivers a consultative, high-signal research artifact to stakeholders before the first calibration call and throughout a search.

## What it does

- **Competitive landscape** — companies competing for the same talent, organized by tier
- **Hiring difficulty heat map** — estimated talent pool sizes and time-to-fill
- **Comp benchmarking** — current market rates by role and level
- **Role-based sourcing engine** — live X-Ray search strings
